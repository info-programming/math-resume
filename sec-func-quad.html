Section 4.4 2次関数

¶

Subsection 4.4.1 2次関数の定義

定義 4.16. 2次関数.
多項式関数のうち、2次式で表される関数を2次関数(quadratic function)という。 すべての二次関数は、次のようにいくつかの形で表せる(\(a, b, c, p, q, \beta, \gamma \in \mathbb{R}, a \ne 0\))。
\begin{align*} f(x) &= a(x-p)^2 + q & (a \ne 0) & (\text{標準形})\\ f(x) &= ax^2 + bx + c & (a \ne 0) & (\text{一般形})\\ f(x) &= a(x-\beta)(x-\gamma) & (a \ne 0) & (\text{因数分解形}) \end{align*}

左辺が標準形の2次関数である\(y = a(x - p)^2 + q\)のグラフは、図4.17のようになる。 この曲線は、物体を上方に投げたときに物体が描く軌道を表せることから放物線(parabola)という。 点\((p, q)\)をこの2次関数の頂点、直線\(x = p\)をこの2次関数の軸(axis)といい、放物線は軸について線対称となる。 また、\(a > 0\)のときは下側に突き出た形なので下に凸(convex)な放物線、\(a < 0\)のときは上側に突き出た形なので上に凸(concave)な放物線という。

2次関数の標準形・一般形・因数分解形の特徴や用途を図4.18に示す。 通常は頂点や最大値・最小値が分かる標準形に変形すると便利なことが多いが、標準形・一般形・因数分解形にはそれぞれ便利な場面があるので、用途に応じて使い分ける必要がある。

Subsection 4.4.2 平方完成

2次関数の形状に関する情報を得るためには、2次関数を標準形で表す必要がある。 そのために、一般形\(f(x) = ax^2 + bx + c\)で表された2次関数を標準形\(f(x) = a(x-p)^2 + q\)に変換する、平方完成(completing the square)という操作を行う。

定理 4.19. 一般形2次関数の頂点と軸.
一般形の2次関数\(f(x) = ax^2 + bx + c\)の頂点は点\((-\frac{b}{2a}, - \frac{b^2 - 4ac}{4a})\)、軸は\(x = -\frac{b}{2a}\)である。
証明.
\begin{gather*} f(x) = ax^2 + bx + c = a(x^2 + \frac{b}{a}x) + c = a(x^2 + \frac{b}{a}x + (\frac{b}{2a})^2) + c - a(\frac{b}{2a})^2 = a(x + \frac{b}{2a})^2 - \frac{b^2 - 4ac}{4a} \end{gather*}

具体的な平方完成の手順は、次のようになる。

アルゴリズム 4.20. 平方完成.
一般形の2次関数\(f(x) = ax^2 + bx + c\)を標準形へと変換するには、次のようにすればよい。
  1. 2次の項と1次の項を\(a\)で括り出す。
  2. 括り出した中で、\(x\)の係数を2で割り、2乗した数を加える。
  3. 加えた数と釣り合いをとるために、加えた数に\(a\)を掛けた数を全体から引く。
  4. 因数分解の公式を用いて\((x - p)^2\)の形の項を作る。

平方完成の例を次に示す。計算が正しいか不安であれば、右から左へ展開して検算するとよい。

例 4.21. 平方完成の例1.
\begin{align*} y &= x^2 - 4x + 1 = (x^2 - 2 \cdot 2x) + 1 = (x^2 - 2 \cdot 2x + 2^2 - 2^2) + 1 = {(x^2 - 2 \cdot 2x + 2^2) - 2^2} + 1 = {(x - 2)^2 - 4} + 1 = (x - 2)^2 - 3\\ y &= x^2 + 3x - 1 = (x^2 + 2 \cdot \frac{3}{2}x) - 1 = (x^2 + 2 \cdot \frac{3}{2}x + (\frac{3}{2})^2 - (\frac{3}{2})^2) - 1 = {(x^2 + 2 \cdot \frac{3}{2}x + (\frac{3}{2})^2) - (\frac{3}{2})^2} - 1 = {(x + \frac{3}{2})^2 - \frac{9}{4}} - 1 = (x + \frac{3}{2})^2 - \frac{13}{4}\\ y &= -x^2 + 3x - 1 = - (x^2 - 3x) - 1 = - (x^2 - 2 \cdot \frac{3}{2}x) - 1 = - (x^2 - 2 \cdot \frac{3}{2}x + (\frac{3}{2})^2 - (\frac{3}{2})^2) - 1 = - {(x^2 - 2 \cdot \frac{3}{2}x + (\frac{3}{2})^2) - (\frac{3}{2})^2} - 1 = - {(x - \frac{3}{2})^2 - \frac{9}{4}} - 1 = - (x - \frac{3}{2})^2 + \frac{9}{4} - 1 = - (x - \frac{3}{2})^2 + \frac{5}{4}\\ y &= 2x^2 + 3x - 1 = 2(x^2 + \frac{3}{2}x) - 1 = 2(x^2 + 2 \cdot \frac{3}{4}x) - 1 = 2(x^2 + 2 \cdot \frac{3}{4}x + (\frac{3}{4})^2 - (\frac{3}{4})^2) - 1 = 2{(x^2 - 2 \cdot \frac{3}{4}x + (\frac{3}{4})^2) - (\frac{3}{4})^2} - 1 = 2{(x - \frac{3}{4})^2 - \frac{9}{16}} - 1 = 2(x - \frac{3}{4})^2 - \frac{9}{8} - 1 = - (x - \frac{3}{2})^2 + \frac{17}{8} \end{align*}
例 4.22. 平方完成の例2.
2次関数\(f(x) = 4x^2 + 6x + 3\)を標準形へ変換する。
\begin{gather*} f(x) = 4x^2+6x+3 = 4(x^2 + \frac{3}{2}x) + 3 = 4(x^2 + \frac{3}{2}x + (\frac{3}{4})^2) + 3 - 4(\frac{3}{4})^2 = 4(x + \frac{3}{4})^2 + \frac{3}{4} \end{gather*}
より、\(f(x)\)の頂点は点\((-\frac{3}{4}, \frac{3}{4})\)、軸は直線\(x = -\frac{3}{4}\)となり、グラフは図4.23のようになる。

Subsection 4.4.3 2次関数の最大・最小

平方完成を行って標準形に変換することは、2次関数の最大・最小問題を解くためにも有効である。

例 4.24. 2次関数の最大値・最小値.
2次関数\(f(x) = 4x^2 + 6x + 3\)の最大値・最小値を求める。
\begin{gather*} f(x) = 4(x + \frac{3}{4})^2 + \frac{3}{4} \geq \frac{3}{4} \end{gather*}
等号は\(x = -\frac{3}{4}\)のとき成り立つので、\(x = -\frac{3}{4}\)のとき最小値\(\frac{3}{4}\)、最大値はない。

第三者が代入して値を検証できるよう、最大値や最小値を求める際は、最大・最小となる\(x\)の値も求める必要がある。

例 4.25. 2次関数の最大値・最小値(定義域が限定されている場合).
2次関数\(f(x) = 4x^2 + 6x + 3 (x \in [0, 10])\)の最大値・最小値を求める。 \(f\)の軸は直線\(x = -\frac{3}{4}\)であるので、\([0, 10]\)で\(f\)は単調増加。 従って、\(x = 0\)で最小値3をとり、\(x = 10\)で最大値463をとる。
例 4.26. 2次関数の最大値・最小値(定義域に文字を含む場合).
2次関数\(f(x) = (x - 7)^2 + 9 = x^2 - 14x + 58 (x \in [m, M]; m < M)\)の最大値・最小値を求める。 区間に文字が含まれているので、\(m, M\)の値により最大・最小となる\(x\)の値が変化することに注意して、場合分けして考えると良い。
  1. \(M \leqq 7\)のとき、\([m, M]\)で\(f\)は単調減少なので、\(x = M\)で最小値\(M^2 - 14M + 58\)、\(x = m\)で最大値\(m^2 - 14m + 58\)をとる。
  2. \(m < 7\)かつ\(7 < M\)のとき、\([m, M]\)に\(f\)の軸が含まれるので、\(x = 7\)で最小値9、\(x = M\)で最大値\(M^2 - 14M + 58\)をとる。
  3. \(7 \leqq m\)のとき、\([m, M]\)で\(f\)は単調増加なので、\(x = m\)で最小値\(m^2 - 14m + 58\)、\(x = M\)で最大値\(M^2 - 14M + 58\)をとる。
以上を整理すると、
\begin{align*} \min f(x) &= \begin{cases} M^2 - 14M + 58 & (M \leqq 7) \\ 9 & (m < 7 < M) \\ m^2 - 14m + 58 & (7 \leqq m) \end{cases}\\ \max f(x) &= \begin{cases} m^2 - 14m + 58 & (M \leqq 7) \\ M^2 - 14M + 58 & (7 < M) \end{cases} \end{align*}

Subsection 4.4.4 2次不等式

\(ax^2 + bx + c > 0\)のような2次不等式は、2次方程式と2次関数の知識を組み合わせて解くと簡単です。

アルゴリズム 4.27. 2次不等式の解法.
\(f(x) = ax^2 + bx + c\)とおくとき、2次不等式\(f(x) > 0\)や\(f(x) < 0\)は次の手順で解く。
  1. 2次方程式\(f(x) = 0\)の解\(x = \beta, \gamma\)を求める。
  2. \(y = f(x)\)のグラフを描く。
  3. \(f(x) > 0\)のときは\(f(x)\)が正になる区間を、\(f(x) < 0\)のときは\(f(x)\)が負になる区間を求める。
例 4.28. 2次不等式の例.
  1. 2次不等式\(x^2 + 5x + 6 > 0\)を解く。 \(x^2 + 5x + 6 = (x + 3)(x + 2) = 0\)の解は\(x = -3, -2\)であるので、\(y = x^2 + 5x + 6\)のグラフは図のようになる。 不等式の解は\(y\)が正となる部分であるので、\(x < -3, -2 < x\)。
  2. 2次不等式\(-4x^2 + 4x + 7 \geqq 0\)を解く。 \(-4x^2 + 4x + 7 = 0\)の解は\(x = \frac{-2 \pm \sqrt{2^2 - 7(-4)}}{-4} = \frac{1 \mp 2\sqrt{2}}{2} (\text{複号同順})\)であるので、\(y = -4x^2 + 4x + 7\)のグラフは図のようになる。 不等式の解は\(y\)が0または正となる部分であるので、\(\frac{1 - 2\sqrt{2}}{2} \leqq x \leqq \frac{1+2\sqrt{2}}{2}\)。
  3. 2次不等式\(x^2 + 4x + 5 > 0\)を解く。 \(x^2 + 4x + 5 = (x + 2)^2 + 1 \geqq 1 (> 0)\)であるので、不等式はすべての\(x \in \mathbb{R}\)について成立する。
  4. 2次不等式\(x^2 + 4x + 5 \leqq 0\)を解く。 \(x^2 + 4x + 5 = (x + 2)^2 + 1 \geqq 1 (> 0)\)であるので、不等式が成立するような\(x\)は存在しない。つまり、この不等式の解はない。
