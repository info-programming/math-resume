Algorithm 4.5. グラフの描き方.

  1. 軸と原点を描く。\(xy\)平面の場合は\(x\)軸と\(y\)軸を書き、それぞれの軸の端に「\(x\)」「\(y\)」と記す。\(x\)軸と\(y\)軸の交点に原点を表す記号\(O\)を記す。

  2. 軸との交点を探し、もし交点があれば交点の座標を書き込む。\(xy\)平面の場合は\(y = 0\)を代入して\(x\)軸との交点を求め、\(x = 0\)を代入して\(y\)軸との交点を求める。

  3. \(x = \pm 1, \pm 2, \pm \frac{1}{2}\)などいくつかの値を代入して、グラフの概形を推測する。

  4. 以上の点を通る線を描く。線上で条件を満たす端点は白丸「○」で、条件を満たさない端点は黒丸「●」で表す。 曲線を描く場合は、「曲がり方が急になる・緩やかになる」「ある値へ近づく」など、どのような曲がり方をしているかを意識し、できるだけ曲線の特徴を反映させて描く。
