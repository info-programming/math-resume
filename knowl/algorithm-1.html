アルゴリズム 2.20. 襷(たすき)掛け.
\(Ax^2+Bx+C\)を\((ax + b)(cx + d)\)の形に因数分解する場合、次のようにする。
  1. 「掛けて\(x^2\)の係数(\(A\))」になる2数\(a, c\)を適当に選び、選んだ2数と\(A\)を縦に並べて記す。
  2. 「掛けて定数項の係数(\(C\))」になる2数\(b, d\)を適当に選び、選んだ2数と\(C\)を(1)で記した右側に縦に並べて記す。
  3. \((\text{左下の数}c) \times (\text{右上の数}b) = bc\)と\((\text{左上の数}a)\times(\text{右下の数}d) = ad\)を、(2)で記した右側に縦に並べて記す。
  4. (3)で記した2数の和(\(ad + bc\))が\(x\)の係数(\(B\))に一致すれば、この\(a, b, c, d\)は正しい組み合わせなので因数分解できる。一致しなければ(1)へ戻る。
\begin{align*} A && C\\ \downarrow && \downarrow\\ a && b && \longrightarrow bc\\ c && d && \longrightarrow ad\\ &&&& B = ad + bc \end{align*}
