例 4.13. 平方完成の例.
2次関数\(f(x) = 4x^2 + 6x + 3\)を標準形へ変換する。
\begin{gather*} f(x) = 4x^2+6x+3 = 4(x^2 + \frac{3}{2}x) + 3 = 4(x^2 + \frac{3}{2}x + (\frac{3}{4})^2) + 3 - 4(\frac{3}{4})^2 = 4(x + \frac{3}{4})^2 + \frac{3}{4} \end{gather*}
より、\(f(x)\)の頂点は点\((-\frac{3}{4}, \frac{3}{4})\)、軸は直線\(x = -\frac{3}{4}\)となり、グラフは図4.14のようになる。
