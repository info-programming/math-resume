Section 1.3 本稿の構成

¶

本稿では、数学における基礎概念を示し、それらの基礎概念を基に展開される理論から得られる知見を示すことを目的とする。

第(((Unresolved xref, reference "ch-sem"; check spelling or use "provisional" attribute)))章から第(((Unresolved xref, reference "ch-media"; check spelling or use "provisional" attribute)))章では、生命や社会の内部における微視的(micro)な現象の理解を目的とする。 ここでは、人間をはじめとする生命の活動を支える情報と、生命からなる社会の活動を支えるコミュニケーション・メディアの関係性を示す。 まず第(((Unresolved xref, reference "ch-sem"; check spelling or use "provisional" attribute)))章で記号と意味を定義し、その対応づけについて述べる。 その上で、第(((Unresolved xref, reference "ch-info"; check spelling or use "provisional" attribute)))章で情報、第(((Unresolved xref, reference "ch-comm"; check spelling or use "provisional" attribute)))章でコミュニケーション、第(((Unresolved xref, reference "ch-media"; check spelling or use "provisional" attribute)))章でメディアの概念を定義し、それぞれの性質を述べた上で分類する。

第(((Unresolved xref, reference "ch-problem"; check spelling or use "provisional" attribute)))章から第(((Unresolved xref, reference "ch-em"; check spelling or use "provisional" attribute)))章では、社会における問題解決の基礎となる方針や概念の理解を目的とする。 ここでは、相互に作用しあう集合であるシステムの概念を基に、システム間で生じる創発や介入といった現象を扱う。 まず第(((Unresolved xref, reference "ch-problem"; check spelling or use "provisional" attribute)))章で問題を定義し、問題解決の方針を示す。 その上で、第(((Unresolved xref, reference "ch-reduction"; check spelling or use "provisional" attribute)))章で還元主義、第(((Unresolved xref, reference "ch-sys"; check spelling or use "provisional" attribute)))章で構造主義、第(((Unresolved xref, reference "ch-em"; check spelling or use "provisional" attribute)))章で創発主義に基づく問題解決を扱う。

第(((Unresolved xref, reference "ch-interference"; check spelling or use "provisional" attribute)))章から第3章では、社会の変革や発展などの巨視的(macro)な現象の理解を目的とする。 第(((Unresolved xref, reference "ch-interference"; check spelling or use "provisional" attribute)))章で他のシステムに対して行う介入の概念を示し、第3章では介入に不可欠な論理を述べる。
