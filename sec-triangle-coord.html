Section 5.3 三角関数と座標

¶

Subsection 5.3.1 三角関数の相互関係

¶

それぞれの三角関数の間には、次の関係式が成り立つ。この関係式は、1つの三角関数の値から他の三角関数の値を求めるときに使うことができる。

定理 5.18. 三角関数の相互関係.
\(\sin^2\theta + \cos^2\theta = 1, \tan^2\theta + 1 = \frac{1}{\cos^2\theta}, \tan\theta = \frac{\sin\theta}{\cos\theta}\)
証明.
  • 原点を中心とする単位円上の点\((\cos\theta, \sin\theta)\)について三平方の定理を用いると、\(\sin^2\theta + \cos^2\theta = 1\)
  • \(\tan\theta = \frac{y}{x} = \frac{r\sin\theta}{r\cos\theta} = \frac{\sin\theta}{\cos\theta}\)
  • \(\sin^2\theta + \cos^2\theta = 1\)の両辺を\(\cos^2\theta\)で割ると、\(\frac{\sin^2\theta}{\cos^2\theta} + 1 = \frac{1}{\cos^2\theta}\)より、\(\tan^2\theta + 1 = \frac{1}{\cos^2\theta}\)

座標平面上での角の関係性を考えると、次の関係式が成り立つことが分かる。

定理 5.19. 三角関数の相互関係(2).
\begin{gather*} \sin(90^\circ - \theta) = \cos\theta, \cos(90^\circ - \theta) = \sin\theta, \tan(90^\circ - \theta) = \frac{1}{\tan\theta}\\ \sin(180^\circ - \theta) = \sin\theta, \cos(180^\circ - \theta) = - \cos\theta, \tan(180^\circ - \theta) = - \tan\theta \end{gather*}

また、三角関数は周期関数のため、次の周期性が成り立つ。

定理 5.20. 三角関数の周期性.
\(\sin(\theta + 2n\pi) = \cos\theta, \cos(\theta + 2n\pi) = \sin\theta (n \in \mathbb{Z})\)

Subsection 5.3.2 三角関数と計量

¶

図形に対して三角関数を用いると辺や角、面積などの大きさを求めることができる。

定理 5.21. 正弦定理.
\(\triangle ABC\)の外接円の半径を\(R\)とおく。
\begin{gather*} \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \end{gather*}
証明.
頂点\(B\)と外接円の中心を結ぶ直線が外接円の円周と交わる点を\(D\)とする。
  1. \(0 < A \leq \frac{\pi}{2}\)のとき、円周角の定理により\(\angle BDC = A\)であり、また\(\angle BCD = \frac{\pi}{2}\)であるので、\(a = 2R\sin A\)となる。
  2. \(\frac{\pi}{2} < A < \pi\)のとき、四角形\(ABCD\)は円に内接するので\(\angle BDC = \pi - A\)であり、また\(\angle BCD = \frac{\pi}{2}\)であるので、\(a = 2R\sin (\pi - A) = 2R\sin A\)となる。
\([1],[2]\)より\(a = 2R\sin A\)である。\(B,C\)についても同様に示せる。

また、次の余弦定理は、直角三角形での三平方の定理を一般の三角形に拡張した定理である。

定理 5.23. 余弦定理.
\(\triangle ABC\)において、\(a = b^2 + c^2 - 2bc\cos A\)。
証明.
頂点\(C\)から\(AB\)に下ろした垂線の足を\(D\)とおく。
\begin{align*} a^2 &= {CD}^2 + {DB}^2 = {CD}^2 + (DA+AB)^2 = (b\sin(\pi - A))^2 + (b\cos(\pi - A) + c) = b^2 + c^2 + 2bc\cos(\pi - A)\\ &= b^2 + c^2 - 2bc\cos A \end{align*}

正弦定理や余弦定理を用いると、三角形の未知の辺や角の大きさを求めることができる。

例 5.24. 正弦定理.
\(\triangle ABC\)において、\(a=\sqrt{2}, b=\sqrt{3}, B=\frac{\pi}{3}\)のとき、\(A\)を求める。 \(\frac{\sqrt{2}}{\sin A} = \frac{\sqrt{3}}{\sin\frac{\pi}{3}}\)より\(\sin A = \frac{1}{\sqrt{2}}\)となるので、\(A = \frac{\pi}{4}, \frac{3\pi}{4}\)。 \(0 < A < \pi - \frac{\pi}{3} = \frac{2}{3}\pi\)より、\(A = \frac{\pi}{4}\)。
例 5.25. 余弦定理.
\(\triangle ABC\)において、\(a=4,b=5,B=\frac{\pi}{3}\)のとき、\(c\)を求める。 \(b^2 = a^2 + c^2 - 2ac\cos B = 16 + c^2 - 4c = 25\)より、\(c^2 - 4c - 9 = 0\)なので、\(c = 2\pm\sqrt{13}\)。 \(c > 0\)であるので、\(c = 2 + \sqrt{13}\)。

また、\(\sin\)を用いて三角形の面積を表すと、次のようになる。

定理 5.26. 三角形の面積.
\begin{gather*} \triangle ABC = \frac{1}{2}bc\sin A \end{gather*}
証明.
頂点\(B\)から\(CA\)へ下ろした垂線の足を\(D\)とすると、\(BD = c\sin A\)であるので、\(\triangle ABC = \frac{1}{2}AC \cdot BD = \frac{1}{2}bc\sin A\)。
